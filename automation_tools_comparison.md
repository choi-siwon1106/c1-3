## 1️⃣ 구현한 워크플로우 개요

### 워크플로우 한 줄 요약:

```
Google Form 제출 → 점수 조건 분기 → Google Sheets 기록 + Discord 알림
```

### 워크플로우 구성 요소

| 구성 요소 | 내용 |
|-----------|------|
| **Trigger** | Google Form 응답 제출 시 |
| **조건 분기 (Filter/Router)** | 점수 기준 합격 / 불합격 분기 |
| **Action 1** | Google Sheets 합격자 명단 기록 (합격 시) |
| **Action 2** | Discord 합격 / 불합격 알림 전송 |

## 3️⃣ 도구별 구현 내용

### 🔵 도구 A – `n8n`

**구현 과정 요약:**

1. Google Sheets Trigger 노드로 폼 응답 시트의 "시험 점수" 탭의 새 행 추가를 감지
2. Code 노드에서 Google Forms 퀴즈 채점 결과 중 0점이 빈 값(" ")으로 들어오는 이슈를 방어 코드로 보정 (빈 값 -> 0 변환)
3. Switch 노드로 점수 기준 3단계 분기 구성 (100점 = 정회원 / 60~99점 = 준회원 / 0~59점 = 불합격)
4. 정회원·준회원 branch : Google Sheets (Append Row)로 각 명단 탭에 기록 후 HTTP Request 노드로 Discord 웹 훅에 합격 알림 전송
5. 불합격 branch : Google Sheets (Append Row)로 재시험 대기 명단에 기록 (7일 뒤 재시험 가능일 자동 계산·저장) -> HTTP Request로 불합격 안내 즉시 발송 -> Wait 노드 (7일 대기) -> HTTP Request로 재시험 가능 안내 발송
6. 전체 흐름을 3가지 케이스 (만점 / 중간 점수 / 저점수)로 테스트하여 각 분기와 알림이 정삭 작동하는지 확인
