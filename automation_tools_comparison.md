## 1️⃣ 구현한 워크플로우 개요

### 워크플로우 한 줄 요약:

```
Google Form 제출 → 점수 조건 분기 → Google Sheets 기록 + Discord 알림
```

### 워크플로우 구성 요소

| 구성 요소 | 내용 |
|-----------|------|
| **Trigger** | Google Form 응답 제출 시 |
| **조건 분기 (Filter/Router)** | 점수 기준 합격 / 불합격 분기 |
| **Action 1** | Google Sheets 합격자 명단 기록 (합격 시) |
| **Action 2** | Discord 합격 / 불합격 알림 전송 |

---

## 2️⃣ 워크플로우 상세 설계

### Google Form 질문 항목
질문	입력 유형
이름	단답형
점수	단답형 (숫자)
합격 기준
항목	내용
만점	100점

### 합격 기준	60점 이상
분기 조건	점수 >= 60 → 합격 / 점수 < 60 → 불합격

### Google Sheets 기록 항목 (합격자만)
열	내용
A열	이름
B열	점수
C열	제출 시간

### Discord 알림 메시지
합격 시: ✅ [홍길동]님 합격을 축하합니다! 점수: 85점   
불합격 시: ❌ [홍길동]님 불합격입니다. 점수: 45점
